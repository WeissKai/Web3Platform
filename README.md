# Web3Platform

Web3Platform 是一个基于Web3技术的校友平台。

## 项目结构

- `LICENSE`: 项目的开源许可证。
- `README.md`: 项目的说明文件。
- `backend/`: 后端代码目录，采用 Node.js。
- `contracts/`: 存放智能合约的目录。
- `frontend/`: 前端代码目录，采用 React。
